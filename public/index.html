NodeBackEnd API - Endpoints

Dit is een eenvoudige overzichtspagina van de beschikbare API endpoints. Gebruik Postman of een browser om deze endpoints te testen.

Afdelingen

  • GET /afdelingen - lijst (optioneel: ?limit=&offset=)
  • GET /afdelingen/search?naam=XYZ - zoek afdelingen op naam
  • GET /afdelingen/:id - details van 1 afdeling
  • POST /afdelingen - maak nieuwe afdeling (body: { naam, leerjaren })
  • PUT /afdelingen/:id - update afdeling (body: { naam, leerjaren })
  • DELETE /afdelingen/:id - verwijder afdeling

Leiding

  • GET /leiding - lijst (optioneel: ?limit=&offset=)
  • GET /leiding/search?voornaam=...&achternaam=...&email=... - zoeken
  • GET /leiding/:id - details van 1 leiding
  • POST /leiding - maak nieuwe leiding (body: { voornaam, achternaam, email, afdeling_id })
  • PUT /leiding/:id - update leiding (body: { voornaam, achternaam, email, afdeling_id })
  • DELETE /leiding/:id - verwijder leiding

Voorbeeld

Voor POST/PUT verzoeken gebruik JSON body en Content-Type: application/json.

Tools

Gebruik Postman voor API tests, TablePlus of DBngin om de database te bekijken en Herd om lokale databases te beheren (of DBngin voor macOS).
